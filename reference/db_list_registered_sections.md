# List registered sections

Lists all sections registered in the master discovery catalog.

## Usage

``` r
db_list_registered_sections(master_path = NULL)
```

## Arguments

- master_path:

  Path to master catalog (uses default if not specified)

## Value

A data.frame with section information

## Examples

``` r
if (FALSE) { # \dontrun{
db_list_registered_sections()
} # }
```
